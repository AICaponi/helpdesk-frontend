HelpMe Sistema de Help Desk

Seja bem vindo ao sistema HelpMe
Esse é um sistema para gerenciamento de chamados de um Help Desk, desenvolvido em Angular 12 e Spring Boot 2.
O sistema está hospedado na nuvem, o frontend se encontra na plataforma Vercel e o backend na plataforma heroku.
O projeto utiliza os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT, REST, boas práticas na construção do código e muitos outros.
